Projects

Learning Logs

Django Web App

Learning Log is a full-stack web app built with Python's Django Web Framework. The app allows users to create new topics, followed by a new entry related to that topic, and the ability to edit an existing entry.

All topics and posts are protected by the django @login_required decorator. Clicking on the topics link on the navbar while not logged in or register will direct the user to the login page.

The app has an admin-panel powered by Django, where the created superuser can add, modify, or delete users, topics and entries.

The app was styled using Bootstrap4.

MicroBlog

Flask Web App

MicroBlog is a Flask web app that serves as a mini blogging platform. Users can create an account/profile, write their own posts, explore other user's posts, follow and unfollow users.

Key Features:
  • Translates a post written in another language dynamically without a page refresh using jquery and utilizing the Microsoft Azure Translation API
  • Features a full-text search bar allowing logged in users to search posts utilizing the Elasticsearch API
  • New users have a randomly avatar generated by the gravatar API
